== Description ==

this plugin allow your Facebook visitor to register in just a click, and synchronize your Facebook notes and WordPress posts and their comments

What's new?
* works without any theme modification, just add the widget
* sync with Facebook links (multiple links per posts)
* login users automatically when connected by Facebook
* create blog users for the connected Facebookers

Why WP-FB?
* let Facebooker register in your site in a second
* bidirectional synchronization notes/posts
* bidirectional synchronization notes/posts comments
* support both user profiles and pages
* automatic Hourly/Daily sync (optional)
* Using Facebook official library
* no more curl (faster and safer)
* support all language
* allow Facebook user to Login and comment in your blog directly
* get commenter Facebook profile pic

do you have any feedback or suggestion? [let us know](http://wordpress.org/support/topic/378928 "Feedback & Suggestions") 

== Installation ==

1. Upload `wp-posts-fb-notes` to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Add the WPFB widget to your sidebar
1. that's all ;)

Configure the plugin:

1. get your AppID and Secret key by [create new facebook application](http://www.facebook.com/developers/createapp.php "create new facebook application").
1. copy the AppID and Secret key from [your app list](http://www.facebook.com/developers/apps.php "your app list")
1. after save the new option connect to Facebook and grand the require permission.
1. that's all :)
